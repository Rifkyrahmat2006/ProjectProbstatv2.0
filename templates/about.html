Tentang Aplikasi

Aplikasi ini dibuat untuk memenuhi tugas Project Probabilitas dan Statistika. Aplikasi ini memiliki fitur-fitur:

  • Visualisasi data menggunakan berbagai jenis grafik (Radar Chart, Box Plot, Histogram, Stem and Leaf Plot)
  • Perhitungan statistik dasar (mean, median, modus, standar deviasi, dll)
  • Analisis distribusi data
  • Penerapan konsep probabilitas
  • Manajemen data peserta olimpiade
