Analisis Statistik OSN

Aplikasi ini dibuat untuk menganalisis data statistik hasil Olimpiade Sains Nasional (OSN) dari berbagai bidang.

Fitur Utama:
  • Analisis data statistik untuk setiap bidang OSN
  • Visualisasi data dalam bentuk berbagai grafik
  • Perbandingan statistik antar bidang
  • Pengelolaan data peserta
Analisis yang Tersedia:
  • Statistik dasar (mean, median, modus, dll)
  • Distribusi frekuensi
  • Visualisasi data:
    • Stem and Leaf Plot
    • Histogram
    • Box Plot
    • Radar Chart (untuk perbandingan)
Pengembang:

Aplikasi ini dikembangkan sebagai bagian dari proyek Probabilitas dan Statistika.
